Atributos HTML

Os atributos HTML fornecem informações adicionais sobre elementos HTML.

  • Todos os elementos HTML podem ter atributos
  • Eles fornecem informações adicionais sobre os elementos
  • Os atributos sao inseridos nas tags de inicio
  • Os atributos vêm em pares, nome="valor";

href

Usado na tag a, define um hyperlink. o atributo href especifica um URL para um endereço

exemplo

Visita o tutorial do w3schools

src

Usado na tag img, para inserir uma imagem no seu documento html

style

O style atributo é usado para adicionar estilos a um elemento, como cor, fonte, tamanho e muito mais.
